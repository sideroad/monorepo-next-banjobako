## Highly recommend
This monorepo is optimize for developer who use vscode

## install
```
yarn
```

## create new app
```
yarn create-app -n hello-world
```

## start up dev server
```
cd apps/hello-world
yarn dev
```

## test
```
cd apps/hello-world
yarn test
```

## generate code from blueprint
```
cd apps/hello-world
yarn gen-component hogehoge # Generate component
yarn gen-page hogehoge # Generate page
```
